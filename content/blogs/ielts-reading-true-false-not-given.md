IELTS Reading 里最容易把人做崩的一类题，就是 **True / False / Not Given**。

很多同学的问题不是读不懂，而是判断标准混在一起：

- 看到题目和原文不完全一样，就选 False
- 找不到完全一样的句子，就选 Not Given
- 原文提到部分信息，就以为等于 True

这类题如果逻辑没分开，做再多也容易反复错。

## 一、先记住最核心的区分

你可以把三种答案理解成下面这张表：

| 选项 | 判断标准 |
| --- | --- |
| True | 原文明确支持题目意思 |
| False | 原文明确反驳题目意思 |
| Not Given | 原文没有给出足够信息判断 |

关键点在于：**False 不是“没看到”，而是“看到了相反信息”**。

## 二、为什么很多人把 False 和 Not Given 搞混

因为做题时大脑会自动补信息。

比如题目说某种政策 “significantly improved student motivation”，原文如果只说 “students participated more often”，这并不一定等于 motivation improved。

这时候最危险的动作就是自己补一句“参与更多应该就是更有动力”，然后误判成 True。

阅读判断题不考你猜测，考的是你能不能只根据原文作答。

## 三、做题顺序应该怎么走

我建议固定用下面这个顺序：

1. 先画出题目关键词
2. 回原文定位对应句
3. 只比较“核心意思”，不要盯字面重复
4. 最后问自己：原文是支持、反驳，还是根本没说

这样做能明显减少“凭感觉选”的情况。

## 四、三个典型误判场景

### 1. 原文只说了一部分

如果题目比原文说得更大、更绝对，往往不能直接选 True。

像 `all`, `always`, `completely`, `only` 这些词，尤其要小心。

### 2. 原文提到同一主题，但不是同一个结论

这通常更接近 Not Given，而不是 False。

### 3. 原文真的给了相反信息

只有这种情况，才是 False。

例如题目说“多数学生偏好线上学习”，原文如果写“most students preferred face-to-face teaching”，这才叫明确反驳。

## 五、怎么练才有效

别只看自己对没对，要记录自己错在了哪一种逻辑。

建议每次练完后把错题分成三类：

- 定位错
- 误把部分信息当完整信息
- 误把 Not Given 当 False

只要你能看见自己的错误模式，正确率会比单纯刷量上升更快。

## 六、结论

True / False / Not Given 这类题的难点，不是词汇本身，而是判断边界。

一句话记住：

- **有依据支持，才是 True**
- **有依据反驳，才是 False**
- **没有足够依据，才是 Not Given**

如果你之后每道题都按这个顺序判断，错误会明显减少。
